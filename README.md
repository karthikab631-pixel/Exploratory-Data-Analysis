# Exploratory Data Analysis (EDA) using Python

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The objective is to understand the data through summary statistics and visualizations to identify patterns, trends, and relationships.

## Objective
- Analyze the dataset using statistical methods.
- Visualize numerical features.
- Identify patterns, trends, and anomalies.
- Draw meaningful insights from the data.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly (Optional)

## Dataset
- Titanic Dataset (loaded using Seaborn)

## Features Performed
- Loaded and explored the dataset
- Displayed dataset information
- Generated summary statistics
- Checked for missing values
- Created histograms
- Created boxplots
- Generated pairplots
- Created a correlation matrix (heatmap)
- Identified patterns and relationships in the data

## Results
- Most passengers were between 20–40 years old.
- Fare distribution is right-skewed.
- The dataset contains missing values and outliers.
- Passengers with higher fares generally had better survival rates.
- Correlation analysis helped understand relationships among numerical features.
